Mini Brazo robótico para actuador neumático mediante ESP32 y BLYNK

Marzo 2026

Introducción

El presente proyecto tiene como objetivo desarrollar un sistema de control para un mecanismo de brazo robótico con un actuador neumático, utilizando el microcontrolador ESP32 como unidad principal de procesamiento.


El sistema es controlado mediante la plataforma Blynk, la cual permite implementar una interfaz de control remoto virtual accesible desde un dispositivo móvil. A través de esta interfaz, el usuario puede interactuar con el sistema en tiempo real.


Como elemento de retroalimentación visual, se incorpora una pantalla OLED que muestra el estado y la secuencia de movimientos del brazo robótico durante su operación.

Descripción del Proyecto

El sistema cuenta con dos modos de funcionamiento: un modo manual, en el cual el usuario controla directamente los actuadores, y un modo semiautomático, donde se ejecuta una secuencia predefinida de movimientos.

Componentes Utilizados

  • Microcontrolador ESP32
  • Módulo de expansión para ESP32 de 30 pines
  • Tres servomotores
  • Un motor paso a paso (28BYJ-48) con driver ULN2003
  • Pantalla OLED (SSD1306)
  • Aplicación móvil Blynk para control remoto
