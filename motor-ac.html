Control de Velocidad PID en motor AC mediante Control de Fase

Abril 2026

Introducción

El presente proyecto tiene como objetivo desarrollar un sistema de control de velocidad de lazo cerrado para un motor de corriente alterna (AC), utilizando el microcontrolador ESP32 como unidad principal de procesamiento. Se implementa un algoritmo PID clásico sin el uso de librerías externas.


El sistema regula la potencia entregada al motor mediante la técnica de control de fase. Para ello, sincroniza un temporizador interno (Hardware Timer) con el cruce por cero de la red eléctrica, calculando el retardo exacto de disparo de un TRIAC.


Como elemento de retroalimentación, se utiliza un encoder que registra los pulsos por segundo (PPS) del motor. La interfaz de usuario se maneja a través del Monitor Serie, donde el usuario puede ingresar el Setpoint deseado y observar la respuesta del sistema en tiempo real.

Descripción del Proyecto

El hardware del proyecto está dividido en la etapa de control (bajo voltaje) y la etapa de potencia (alto voltaje de la red eléctrica). El controlador ajusta dinámicamente un ángulo de retardo (delay en microsegundos) para mantener la velocidad constante ante perturbaciones mecánicas.

Componentes Utilizados

  • Microcontrolador ESP32
  • Módulo de expansión para ESP32 de 30 pines
  • Motor AC vn-382
  • Módulo Dimmer AC 1 Canal 8A 110V/220V
  • Sensor de velocidad con encoder
